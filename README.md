# DevOps

> DevOps 是近年來不斷在興起的一種軟體開發概念。DevOps 即是 Development 和 Operations 的結合，顧名思義主要是包含了開發者（包含軟體品質、測試人員）與維運人員（包含維護、系統架設人員）的之間的一種資訊文化運動。

$$
Development -And- Operations
$$

{% hint style="success" %}
DevOps 的核心概念其實包含了四種元素：

* Culture
* Automation
* Measurement
* Sharing
{% endhint %}
